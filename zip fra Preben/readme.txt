Experior 6:
-Install Experior (6.0.18199 or later)
-Copy all files in  the "Experior 6" folder to Experior installation folder (default: C:\Xcelgo\Xcelgo Experior 6)

Experior 7:
-Install Experior (7.0.18225 or later)
-Copy all files in  the "Experior 7" folder to Experior installation folder (default: C:\Xcelgo\Xcelgo Experior 7)

Robot Studio 2019.4:
-Copy all files in the folder "RobotStudio addins" to RobotStudio addin folder (default: C:\Program Files (x86)\ABB\RobotStudio 2019\Bin\Addins)

Add new robots to Experior:
-Experior 6.0: Create folder C:\Xcelgo\Xcelgo Experior 6\Robots
-Experior 7.0: Create folder C:\Xcelgo\Xcelgo Experior 7\Robots
-Export collada file from Robot Studio and place the file in this folder. 
-Start Experior

NOTE:
-In Robot Studio grippers or tools must be a "Robot Studio Tool"
-Signals supported: DigitalInput, DigitalOutput, AnalogOutput, AnalogInput, GroupInput, GroupOutput
-The graphics for the "Simple gripper" is found in the folder "Simple gripper graphics"
